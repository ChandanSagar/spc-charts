NetApp ANF CSI Driver for AKS

To upgrade an install through helm:

helm upgrade --install anf-csi-aks-driver ./anf-csi-aks-driver \
    --namespace kube-system \
    --set azurecreds.subscriptionid=SUBSCRIPTION_ID \
    --set azurecreds.tenantid=TENANT_ID \
    --set azurecreds.clientid=CLIENT_ID \
    --set azurecreds.clientsecret=CLIENT_SECRET \
    --set sc.region=REGION \
    --set sc.resourcegroup=RESOURCE_GROUP \
    --set sc.netappaccount=NETAPP_ACCOUNT \
    --set sc.capacitypool=CAPACITY_POOL \
    --set sc.virtualnetwork=VIRTUAL_NETWORK \
    --set sc.subnet=SUBNET
